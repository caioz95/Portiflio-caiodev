Projeto de CRUD em Java com Servlet

Este projeto é um sistema de gerenciamento de funcionários em Java com Servlet, conectando-se a um banco de dados MySQL. Ele oferece operações básicas de CRUD (Create, Read, Update, Delete) e é licenciado sob a MIT License.

CRUD em Java Swing

O projeto cria um sistema de autenticação avançado em Java Swing com banco de dados, oferecendo login, verificação de credenciais, tela principal, registro de usuário e recuperação de senha. Utiliza Java para a lógica, Swing para a interface e bancos de dados como MySQL ou PostgreSQL. Os benefícios incluem um sistema robusto e seguro e a oportunidade de aprender tecnologias relevantes. Os objetivos são eficiência, segurança e boa experiência do usuário, além de aprofundar conhecimentos em Java Swing e banco de dados.

Lógica de Programação

Este projeto concluído é o Jogo da Sorte, desenvolvido em Java. Os jogadores tentam adivinhar um número escolhido aleatoriamente pela máquina dentro de um intervalo definido, recebendo dicas para orientar suas suposições. Utiliza Java (Versão 8) e Eclipse IDE. Os jogadores têm três tentativas para acertar o número sorteado. Este projeto está licenciado sob a MIT License e aceita contribuições.
